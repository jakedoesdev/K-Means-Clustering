# project 2
Zach Bender, Jacob Everett, Devon Gates, Mitchell Koen

About: This program performs kmeans and hierarchical clustering and calculates the Dunn Index on any number of x y coordinates given a user-defined configuration file and points file.

COMPILATION INSTRUCTIONS:

A makefile is provided. Type "make" and an output file will be created called "output" i.e. ./output to run. 
When the program runs, you will be prompted to enter the filename of the config file. Type it in and press enter and
calculations will be run.
